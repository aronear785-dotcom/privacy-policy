Privacy Policy

This application is designed to assist teachers in creating exam reports, grade books, and merging PDF files locally on the user’s device.

1. Data Collection

This app does not collect, store, or transmit any personal data to external servers.

All operations, including processing of Excel files, grade calculations, and PDF merging, are performed locally on the user’s device.

The application does not require user registration or account creation.

2. Data Usage

All data provided by the user (such as Excel files containing student or teacher information) is used solely for processing within the app and is not accessed, uploaded, or stored externally.

3. Storage Permissions (Important)

The app requires access to device storage in order to:

  • Read Excel files selected by the user for generating reports and grade sheets
  • Access PDF files for merging and processing
  • Save generated reports and merged PDF files on the user’s device

All file operations are performed locally on the device.

The app does not access any files without user action and does not upload or share any data externally.

4. Third-Party Services

This application uses Google AdMob to display advertisements.

Google AdMob may collect and process certain non-personal data, such as:

  • Advertising ID
  • Device information
  • Usage data (for ad personalization and analytics)

For more information, please refer to:

https://policies.google.com/privacy

5. Data Sharing

We do not sell, share, or transfer any personal data to third parties.

User data remains entirely on the device and is never transmitted to external servers controlled by us.

6. Security

Since all data processing is performed locally, no personal data is stored on external servers. Users retain full control over their files.
